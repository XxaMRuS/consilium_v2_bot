## 💪 ТРЕНИРОВКИ (УПРАЖНЕНИЯ)

### 1. 🏋️ Спорт → 💪 Упражнения

```
🏋️ УПРАЖНЕНИЯ
├── 👶 Новичок (упражнения для начинающих)
├── 🤓 Профи (сложные упражнения)
└── ⏱ На время / 🔢 Повторы / 🏋️ Вес / 🏃 Дистанция
```

### 2. Выполнение упражнения:

```
ШАГ 1: Выберите упражнение
        ↓
ШАГ 2: Введите ваш результат
        - Для повторов: число (например: 50)
        - Для времени: секунды (например: 90 = 1:30)
        - Для веса: кг (например: 75.5)
        ↓
ШАГ 3: Обязательно прикрепите доказательство:
        - 📷 Фото
        - 🎥 Видео файл
        - 📄 Документ
        - 🔗 Ссылка на видео (YouTube, Vimeo)
        ↓
Готово! ✅ Упражнение выполнено!
```

### Что происходит:
- 📊 Результат сохраняется в базу данных
- ⭐ Вам начисляются очки
- 📈 Вы попадаете в рейтинг TOP-20
- 📢 В канал отправляется уведомление о вашем достижении
- 🏆 Если вы побили рекорд - получаете дополнительные очки!

```
│   └── ✅ Все упражнения теперь участвуют в соревнованиях!
│

│   └── ❌ Соревнования отключены для всех упражнений!
│
├── 🟡 Включить новичковые
│   └── ✅ Только упражнения уровня "beginner" участвуют!
│
└── ⏬ Список упражнений
    ├── 📋 Все упражнения с статусом соревнований
    │
    ├── Упражнение 1: Отжимания
    │   ├── 🏆 Очки за победу: 5,000
    │   ├── 📊 Банк: 10,000
    │   └── ✅ Статус: Включено
    │
    ├── Упражнение 2: Приседания
    │   ├── 🏆 Очки за победу: 3,000
    │   ├── 📊 Банк: 6,000
    │   └── ✅ Статус: Включено
    │
    └── [Все упражнения с деталями]
```

### Как это работает:
1. **🏁 Соревнования на скорость** - кто быстрее выполнит упражнение
2. **🏆 Банк очков** - pool очков для победителей (настраивается для каждого упражнения)
3. **🥇 Распределение банка:**
   - 1 место: 50% от банка
   - 2 место: 30% от банка
   - 3 место: 20% от банка

### Массовые действия:
- **🟢 Включить все** - включает соревнования для ВСЕХ упражнений сразу
- **🔴 Выключить все** - выключает соревнования для ВСЕХ упражнений сразу
- **🟡 Включить новичковые** - включает только для упражнений уровня "beginner"
