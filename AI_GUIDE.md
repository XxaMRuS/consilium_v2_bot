# 🤖 AI-ТРЕНЕР - ИНСТРУКЦИЯ

## 🎯 ЧТО ДОБАВЛЕНО:

### 1. 💬 **Умный тренер (YandexGPT)**
- Персональные советы по тренировкам
- Ответы на вопросы о фитнесе
- Мотивация и рекомендации

### 2. 📸 **Анализ фото (Gemini)**
- Анализ техники выполнения упражнений
- Конкретные советы по улучшению
- Оценка от 1 до 10

### 3. ⚡ **Мгновенные советы (Groq)**
- Быстрые рекомендации на каждый день
- Мотивация в реальном времени
- Меньше 1 секунды ответ

### 4. 📊 **Анализ прогресса (Groq)**
- Глубокий анализ тренировок
- Выявление паттернов
- Персональные рекомендации

### 5. 📅 **Календарь тренировок**
- Визуальное отображение тренировок по дням
- Навигация по месяцам
- Статистика активности

---

## 🚀 КАК ИСПОЛЬЗОВАТЬ:

### В главном меню:
1. Нажми **🏋️ Спорт**
2. Выбери нужную функцию:
   - **📅 Календарь** - просмотр тренировок по дням

### В AI-меню:
1. Нажми **🤖 AI Тренер**
2. Выбери нужную функцию:
   - 💬 **Спросить тренера** - задай вопрос
   - 📸 **Анализ фото** - отправь фото упражнения
   - ⚡ **Быстрый совет** - получи рекомендацию
   - 📊 **Анализ прогресса** - проанализируй свои результаты

---

## 🔑 API КЛЮЧИ:

### Для локального тестирования:
Добавь в `.env` файл (ключи есть у владельца бота)

### Для Render:
Добавь в **Environment Variables**:
- `YANDEX_API_KEY`
- `YANDEX_FOLDER_ID`
- `GEMINI_API_KEY`
- `GROQ_API_KEY`
- `OPENROUTER_API_KEY`

---

## ⚠️ ВАЖНО:

### Система fallback (автоматическое переключение):
Бот использует **только бесплатные API** с автоматическим переключением:

1. **Groq** (основной) - самый быстрый
2. **OpenRouter** (бесплатные модели) - резервный
3. **YandexGPT** (резерв) - если другие недоступны

### Лимиты API:
- **Groq**: Очень быстрый, рекомендуем для частых запросов
- **OpenRouter**: Бесплатные модели (Gemma, Llama, Mistral)
- **YandexGPT**: Ограничен, используем как резерв
- **Gemini**: Быстрый для анализа фото

### Рекомендации:
1. **Groq** для ежедневных советов и анализа прогресса (быстрый)
2. **Yandex** для глубоких вопросов (качественный)
3. **Gemini** для анализа фото (мультимодальный)
